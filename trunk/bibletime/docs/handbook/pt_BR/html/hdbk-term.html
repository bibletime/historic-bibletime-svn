Como iniciar o BibleTime

Iniciando BibleTime

BibleTime um arquivo executável que é integrado ao desktop. Você pode iniciar o BibleTime do menu de início com esse ícone:

BibleTime pode ser iniciado de um terminal de prompt de comando. Para iniciar o BibleTime, abra uma janela do terminal e digite: 

bibletime

Personalização de início

De um terminal você pode usar o BibleTime para abrir um versículo aleatório na bíblia padrão: 

bibletime --open-default-bible "<random>"

Para abrir numa passagem como João 3:16, use: 

bibletime --open-default-bible "João 3:16"

Você também pode usar os nomes dos livros em seu idioma de nomes atual.
