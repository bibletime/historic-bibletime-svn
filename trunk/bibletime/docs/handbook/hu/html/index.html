Kivonat

A BibleTime egy teljesen ingyenes és szabad Biblia tanulmányozó program. A program kezelőfelülete a Qt keretrendszerre épült, ami lehetővé teszi többféle operációs rendszeren való futtatását, mint Linux, Windows, FreeBSD és Mac OS X rendszeren. A szoftver a SWORD programozási könyvtárat használja, több, mint 200 ingyenes Bibliaváltozattal, kommentárokkal, szótárakkal és könyvekkel, amit a Crosswire Bible Society biztosít.
