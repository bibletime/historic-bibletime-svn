Eszköztár leírás

Kérjük nézze meg az előző részt az eszköztár gombjainak magyarázatához. Lehetősége van az egérmutatót a gomb fölé vinni, és néhány másodpercen belül megjelenik egy buborék súgó egy rövid leírással.

Szükségeinek megfelelően beállíthatja az eszköztárat, ehhez használja a következő menüpontot:

Settings → Configure toolbars

Eszköztárak testreszabása

A részletes információkért tekintse meg ezt a részt.
